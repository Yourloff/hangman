## Pre-launch preparation

```
bundle install
```

## Usage:

```
bundle exec ruby main.rb
```

To add new words to the game:

`data/words.txt`. Add Cyrillic uppercase words in one line.

The library is used [colorize](https://github.com/fazibear/colorize).
